# Task 3

## Overview

This task has two parts:

1. Create a custom dataset with 5 columns, 15 rows, and a custom index.
2. Explore, clean, and analyze the Titanic dataset.

## Folder Structure

- data/
  - dataset-2.csv (Titanic dataset)
  - custom_dataset.csv (generated custom dataset)
- src/
  - part1_custom_dataset.py
  - part2_titanic.py

## How to Run

From the task-3 folder:

```bash
python src/part1_custom_dataset.py
python src/part2_titanic.py
```

## Notes

- Part 1 saves the custom dataset to data/custom_dataset.csv.
- Part 2 reads data/dataset-2.csv and prints the required exploration, cleaning, and analysis steps.
