# CLI Programming Tasks

This project implements one task from each level, with each task in its own file:

- Level 1: Check if a number is prime ([tasks/prime_task.py](tasks/prime_task.py))
- Level 2: Count the number of vowels in a string ([tasks/vowel_count_task.py](tasks/vowel_count_task.py))
- Level 3: Build a Tic-Tac-Toe (X/O) game (CLI-based) ([tasks/tic_tac_toe_task.py](tasks/tic_tac_toe_task.py))

## How to Run

Run any task file directly:

```bash
python3 tasks/prime_task.py
python3 tasks/vowel_count_task.py
python3 tasks/tic_tac_toe_task.py
```

## Notes

- The prime check uses an optimized trial division approach.
- The vowel counter treats a, e, i, o, u as vowels.
- Tic-Tac-Toe alternates turns until there is a winner or draw.
